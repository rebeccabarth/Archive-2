    Beccas Events Process

    Every one of our events follows the outlined Beccas Events Process:

    1. Step 1: Objectives
      • - Event Brief and Proposal
    2. Step 2: Draft Outline Plan and Detailed Planning
      • - Target Dates, Venue, Staffing and Costs
      • - Identify Budget, Operations, Logistics and Event Concerns
    3. Step 3: Organising the Event
      • - Event Preparations, Stakeholder Liasion
    4. Step 4: Executing the Event
      • - Manage Operations and Run the Event
    5. Step 5 : Conclusion
    • - Prepare Event Evaluation and Collect Feedback
